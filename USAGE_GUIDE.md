# 🚀 ScreenDemo 安装与使用指南 (v2.1.0)

感谢您选择 ScreenDemo！这是一个专为 Windows 11 设计的高效截图与记录管理工具。本文档将指引您完成安装，并帮助您快速上手核心功能。

---

## 📥 第一步：下载与安装

### 1. 获取安装包
您可以从项目的 Release 页面下载以下任一版本：
- **安装版 (Setup.exe)**：推荐使用，包含完整的安装与卸载程序。
- **绿色版 (Portable.exe)**：无需安装，单文件运行，适合临时使用。
- **压缩版 (.zip)**：解压即用，包含完整的程序文件夹。

### 2. 安装过程
1. 双击 `ScreenDemo Setup 2.1.0.exe`。
2. **安全提示解决**：由于本程序未进行商业签名，Windows SmartScreen 可能会拦截。
   - 点击 **“更多信息” (More info)**。
   - 点击右下角的 **“仍要运行” (Run anyway)**。
   【图片内容：Windows SmartScreen 蓝色警告窗口及操作指引】
3. 选择安装目录（默认为 Program Files）。
4. 完成安装后，程序会自动启动并缩小到**系统托盘**（屏幕右下角图标区）。

## ✨ 核心功能介绍

### 1. 智能截图与自动保存
- **快捷键触发**：按下 `Ctrl + Shift + X`（可在设置中修改）。
- **多屏适配**：程序会自动判断您鼠标所在的显示器，并截取该屏幕的全屏画面。
- **自动采集**：截图的同时，程序会自动读取剪贴板中的文本或图片并关联保存。

### 2. 智能操作浮窗
截图完成后，鼠标附近会弹出一个磨砂玻璃质感的浮窗：
- **快速备注**：直接在浮窗输入文字，内容会实时自动保存。
- **内容预览**：即时查看截图缩略图和剪贴板内容。
【图片内容：截图后出现的智能操作浮窗界面】

### 3. 智能侧边栏 (Sidebar Drawer)
这是 v2.1.0 的核心特性：
- **极简把手**：Memo 窗口平时会隐藏在屏幕边缘，仅露出一小条把手。
- **点击展开**：左键单击把手，窗口会顺滑滑出。
- **位置记忆**：支持右键按住把手自由移动位置，并永久记忆您的布局方案。

## 🛠️ 详细使用教程

### 如何管理您的记录 (Memo)
1. **打开 Memo**：点击浮窗底部的 "Open Memo Window" 或右键点击托盘图标选择 "Memo Window"。
2. **状态切换**：点击每条记录左侧的复选框，可将其标记为 "Todo" 或 "Done"。
3. **筛选查看**：使用顶部的标签页快速切换查看全部、待办或已完成的记录。
4. **拖拽排序**：长按并拖动记录，可自由调整它们的优先级顺序。
5. **双击编辑**：双击任意一条记录，可重新进入编辑模式修改笔记。

### 如何自定义设置
1. 右键点击托盘图标，选择 **Settings**。
2. **快捷键修改**：点击快捷键输入框，直接按下您想设定的新组合键（如 `Alt + A`）。
3. **保存路径**：点击“浏览”选择您希望存储截图的本地文件夹。
4. **侧边栏配置**：
   - 调整把手的宽度和高度。
   - 设置展开后的窗口默认宽度。
   - 选择侧边栏固定在哪个屏幕（跟随鼠标或固定主屏）。

## ❓ 常见问题与解决

### Q1：按下快捷键没有反应？
- **原因**：可能是快捷键被其他软件（如 QQ、微信或 Snipping Tool）占用。
- **解决**：进入 Settings 界面，更换一个新的快捷键组合。

### Q2：为什么我的截图文件夹是空的？
- **原因**：如果您手动修改了保存路径，请确保该目录具有写入权限。
- **解决**：在 Settings 中重置保存路径，或检查文件夹属性。

### Q3：把手挡住了我的其他操作？
- **解决**：
  - 方法一：右键按住把手，将其拖拽到屏幕不常用的位置（如靠上或靠下）。
  - 方法二：在 Settings 中减小 "Handle Width"（建议 5-10px）。

### Q4：程序运行变慢或报错？
- **解决**：查看项目根目录下的 `logs/` 文件夹，将对应的日志文件反馈给开发人员。
